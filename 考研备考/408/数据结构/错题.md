### 绪论部分
关于数据结构的说法中正确的是：**数据的逻辑结构独立于数据的存储结构**
同一个逻辑结构--->不同的底层实现

### 时间复杂度部分
```cpp
int sum = 0;
for(int i = 1; i < n;i*=2)
{
	for(int j = 0;j < i ;j++)
	{
		//something
	}
}
```
上面算法的时间复杂度为O(n)
举例子:当n = 16的时候
注意这里是小于号，不是小于等于号！
执行如下：
1->.
2->..
4->....
8->........
到此为止，相当于进行了15次操作，时间复杂度为O(n)

### 分析斐波那契数列的时间复杂度：

$$F(n) = \begin{cases} 0 & n = 0 \\ 1 & n = 1 \\ F(n-1) + F(n-2) & n > 1 \end{cases}$$
使用递归方式求解F(5) ，对应的二叉树的高度为5 ， 节点数为
$$
2^5-1
$$
时间复杂度是$$O(2^n)$$



### 空间复杂度的计算的时候
函数递归调用栈分两种情况：
- 空间开销与递归深度无关，即每次递归的开销是常量，O(n)
- 空间开销与深度相关，这时候需要进行计算
