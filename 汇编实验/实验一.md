
## 为什么有LF 格式与CRLF格式的区别？

这源于早期的机械打字机：  
当打完一行需要换行时，你需要做**两个动作**：

1. 把打字头推回最左边（**回车 Carriage Return**，`\r`）。
2. 把滚筒卷一下让纸往上走一格（**换行 Line Feed**，`\n`）。

后来到了计算机时代：

- **Windows** 依然保留了打字机的两步操作，严格使用 `\r\n`。
- **Linux/Unix** 的开发者觉得多此一举，既然换行了，默认光标肯定回到行首啊，于是简化为只使用 `\n`。


## 脚本的编写相关

```
#! /bin/bash
g++ -c lexer.cpp -o lexer.o 
g++ -c parser.cpp -o parser.o 
g++ -c main.cpp -o main.o 

g++ lexer.o parser.o main.o -o Main -lm

```

过滤源程序中的空白符(空格，注释、tab，回车、换行等)
识别关键字、标识符、数字以及运算符。
